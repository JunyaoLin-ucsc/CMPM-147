Description

This experiment generates a suspicious restaurant menu by creating random text based on a grammar. The code was originally developed on Glitch and then migrated to this static webpage.

Technical

The generator uses a simple grammar-based text generation approach. It randomly selects elements from predefined word arrays to replace the placeholders in a template string. jQuery is then used to display the generated text in the webpage.

Reflection

By integrating the Glitch project into a local static website, I learned how to transition and adapt code for different environments. This process improved my understanding of DOM manipulation with jQuery and managing multi-file projects. Having the final result match the original Glitch style was also a key goal.
